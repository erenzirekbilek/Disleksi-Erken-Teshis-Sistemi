# FERPA Compliance Checklist - Dyslexia Early Detection System

## Overview

This document outlines the compliance requirements for the Dyslexia Early Detection System under the Family Educational Rights and Privacy Act (FERPA).

---

## 1. Definition of Educational Records

### Covered Records
- [x] Student identification data
- [x] Assessment results
- [x] Analysis outputs (risk scores)
- [x] Teacher observations linked to student

### Not Covered (if applicable)
- [ ] Law enforcement unit records
- [ ] Employment records
- [ ] Medical records (separate HIPAA considerations)
- [ ] Notes by teachers not shared with others

---

## 2. Eligible Students

### Definition
- [x] Students in K-12 schools
- [x] Students aged 5-18 (target age group)

### Rights Transfer
- [x] Rights transfer to parent for minors
- [x] Rights transfer to student at 18

---

## 3. Consent Requirements

### Consent vs. Disclosure Without Consent

**May Disclose Without Consent:**
- [ ] School officials with legitimate educational interest
- [ ] Other schools where student transfers
- [ ] Judicial order or subpoena
- [ ] Health and safety emergencies

**Requires Consent:**
- [x] Research purposes (even if anonymized)
- [x] Commercial purposes
- [x] Disclosures to outside parties

### Implementation
- [x] Parental consent form created
- [x] Consent specifies data types
- [x] Consent specifies purpose
- [x] Consent specifies duration
- [x] Withdrawal process defined

---

## 4. Directory Information

### Definition
- [x] Information that would not be harmful if disclosed
- [x] Must be designated by school

### Our Policy
- [x] We do NOT use directory information
- [x] All data is protected/encrypted

---

## 5. Disclosure Rules

### 5.1 Disclosure to School Officials

**Legitimate Educational Interest (must document):**
- [x] Performing a task within educational institution
- [x] Direct control over educational records
- [x] Specific task documented

**Our Implementation:**
- [x] Define school official roles
- [x] Document access controls
- [x] Log all access

### 5.2 Disclosure to Third Parties

**Prohibited without consent:**
- [x] No disclosure to external researchers
- [x] No disclosure to commercial entities
- [x] No disclosure for advertising

**Exceptions (not applicable):**
- [ ] Health/safety emergency
- [ ] Judicial order
- [ ] Transfer to another school

---

## 6. Parental Rights

### Rights Afforded to Parents
- [x] Right to inspect and review records
- [x] Right to request amendment
- [x] Right to consent to disclosures
- [x] Right to file complaint

### Implementation
- [x] Request portal for record access
- [x] Amendment request process (30 days)
- [x] Complaint procedure (DPA contact)

---

## 7. Record Keeping

### Disclosure Log (Section 99.31)
- [x] Track all disclosures
- [x] Record: who, what, when, purpose
- [x] Retain for duration of enrollment + 3 years

### Access Log
- [x] Log all access to records
- [x] Include: user, time, action
- [x] Retain for 7 years

---

## 8. Security Requirements

### Physical Safeguards
- [x] Secure facility access
- [x] Locked storage for physical records
- [x] Secure destruction procedures

### Technical Safeguards
- [x] Access controls (unique IDs)
- [x] Encryption
- [x] Audit trails
- [x] Secure transmission (TLS)

---

## 9. Educational Agency Definition

### Applicable Agencies
- [x] Public schools (K-12)
- [x] School districts
- [x] State education agencies

### Application
- [x] Signed agreement with school district
- [x] Research designation documented
- [x] Institution becomes "school official"

---

## 10. Research Exception (Section 99.31)

### Requirements for Disclosure
- [x] Educational purpose
- [x] Direct control over records
- [x] Legitimate educational interest
- [x] Use limited to stated purpose

### Additional Safeguards
- [x] Data anonymized before research use
- [x] No re-identification
- [x] No secondary use

---

## 11. Parent Access Implementation

### Access Request Process
1. Parent submits written request
2. Verify identity (security questions)
3. Gather records (10 days)
4. Review for exemptions (5 days)
5. Provide access or written explanation

### Fees
- [ ] May charge for copying (not exceed actual cost)
- [x] No charge for review
- [x] No charge for electronic delivery

---

## 12. Amendment Process

### Request Requirements
- [x] Written request required
- [x] Specify what to amend
- [x] Provide reason

### Response Requirements
- [x] Decide within 45 days
- [x] Inform of decision
- [x] If denied: notify of right to hearing

### Hearing Process
- [x] Conduct hearing
- [x] Decision based on accuracy
- [x] If denied: notify of right to add statement
